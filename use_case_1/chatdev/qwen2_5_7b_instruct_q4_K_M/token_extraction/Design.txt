### Getting Started with Python

1. **Installation**:
   - First, you need to install Python on your computer. You can download it from the official Python website: https://www.python.org/downloads/
   - Make sure to check the box that says "Add Python to PATH" during installation if you're using Windows.

2. **Setting Up an IDE (Integrated Development Environment)**:
   - For beginners, I recommend starting with a simple text editor or an integrated development environment (IDE) like PyCharm, Visual Studio Code, or even just a basic text editor like Notepad++.
   - You can also use online Python environments like Repl.it if you prefer not to install anything on your computer.

3. **First Program**:
   - Let's start with a simple "Hello, World!" program. Open your chosen IDE and create a new file. Save it as `hello_world.py`.

Here’s the code:

```python
# This is a comment in Python

print("Hello, World!")
```

- Run the script by clicking on the run button in your IDE or by typing `python hello_world.py` in your command line if you have Python installed.

4. **Basic Syntax**:
   - Python uses indentation to define blocks of code. Unlike languages like Java or C++, Python doesn't use curly braces `{}`.
   - Example:

     ```python
     x = 5
     y = 10

     # Indentation is important in Python!
     if x < y:
         print("x is less than y")
     else:
         print("x is not less than y")
     ```

5. **Variables and Data Types**:
   - Python has several built-in data types: integers, floats, strings, lists, dictionaries, etc.

   ```python
   # Integer
   a = 10

   # Float
   b = 3.14

   # String
   name = "Alice"

   # List
   fruits = ["apple", "banana", "cherry"]

   # Dictionary
   person = {"name": "Bob", "age": 25}
   ```

6. **Control Structures**:
   - If you want to control the flow of your program, use conditional statements and loops.

   ```python
   x = int(input("Enter a number: "))

   if x > 0:
       print("The number is positive")
   elif x < 0:
       print("The number is negative")
   else:
       print("The number is zero")
   ```

7. **Functions**:
   - Functions are blocks of organized, reusable code that can take inputs and produce outputs.

   ```python
   def greet(name):
       return f"Hello, {name}"

   message = greet("Alice")
   print(message)
   ```

8. **Lists and Tuples**:
   - Lists are mutable (can be changed) and tuples are immutable.

   ```python
   # List
   numbers = [1, 2, 3]
   numbers.append(4)  # You can modify it

   # Tuple
   point = (10, 20)
   ```

9. **Dictionaries**:
   - Dictionaries are used to store data values in key: value pairs.

   ```python
   student = {
       "name": "Tom",
       "age": 20,
       "courses": ["Math", "Physics"]
   }
   print(student["name"])
   ```

10. **Modules and Packages**:
    - Python comes with a large standard library, and you can extend its functionality by using third-party packages.

    ```python
    import math

    print(math.pi)
    ```

- Try to write small programs that solve simple problems or simulate real-world scenarios. For example, create a program that calculates the area of a circle given the radius.

#### Exercise 1: Calculate the Area of a Circle

Write a program that takes the radius of a circle as input and calculates its area.

```python
import math

def calculate_area(radius):
```

#### Exercise 2: Simple Calculator

Create a simple calculator that performs addition, subtraction, multiplication, and division.

```python
def add(x, y):
    return x + y

def subtract(x, y):
    return x - y

def multiply(x, y):
    return x * y

def divide(x, y):
    if y != 0:
        return x / y
    else:
        return "Division by zero is not allowed"

print("Select operation:")
print("1. Add")
print("2. Subtract")
print("3. Multiply")
print("4. Divide")

choice = input("Enter choice (1/2/3/4): ")

num1 = float(input("Enter first number: "))
num2 = float(input("Enter second number: "))

if choice == '1':
    print(f"{num1} + {num2} = {add(num1, num2)}")
elif choice == '2':
    print(f"{num1} - {num2} = {subtract(num1, num2)}")
elif choice == '3':
    print(f"{num1} * {num2} = {multiply(num1, num2)}")
elif choice == '4':
    result = divide(num1, num2)
    if isinstance(result, str):
        print(result)
    else:
        print(f"{num1} / {num2} = {result}")
else:
    print("Invalid input")
```
