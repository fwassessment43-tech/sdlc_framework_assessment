## Original Requirements

Create a snake game in Python

## Product Goals

- Ensure an engaging user experience with smooth animations
- Make the game responsive and easy to navigate
- Implement a clean, intuitive UI

## User Stories

- As a player, I want to control the snake using arrow keys
- As a player, I want to see my score increase as I eat food
- As a player, I want to restart the game when I lose
