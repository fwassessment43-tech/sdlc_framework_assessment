## Implementation approach

We will implement a simple snake game using Python with Pygame library for graphics and user input handling. The game will feature basic snake movement, food generation, collision detection, score tracking, and a game over screen with restart functionality.

## File list

- main.py
- snake_game.py

## Program call flow


sequenceDiagram
    participant U as User
    participant SG as SnakeGame
    participant S as Snake
    participant F as Food
    U->>SG: Start game
    SG->>S: Initialize snake
    SG->>F: Generate food position
    loop Game Loop
        U->>SG: Handle events (key presses)
        SG->>S: Update direction
        S->>S: Move snake
        S-->>SG: Return new head position
        SG->>S: Check for collision with self or walls
        alt Snake eats food
            S->>F: Check if snake head at food position
            F-->>S: Yes, return True
            S->>S: Grow snake
            SG->>F: Generate new food position
        end
        SG->>SG: Update score
        SG->>SG: Draw game elements
    end
    alt Game Over
        U->>SG: Handle events (key presses)
        SG-->>U: Show game over screen with restart option
    end
