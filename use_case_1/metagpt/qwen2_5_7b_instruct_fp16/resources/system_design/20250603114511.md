## Implementation approach

We will utilize Pygame for creating a smooth and engaging snake game. Pygame is a popular library in Python that provides functionalities for handling graphics, sound, and input which are essential for developing games like Snake.

## File list

- main.py
- game.py

## Data structures and interfaces


classDiagram
    class Game {
        -screen: pygame.Surface
        -clock: pygame.time.Clock
        -snake: Snake
        -food: Food
        -score: int
        +__init__(width: int, height: int)
        +run()
        +update()
        +draw()
    }
    class Snake {
        -body: List[Tuple[int, int]]
        -direction: Tuple[int, int]
        +__init__(start_pos: Tuple[int, int])
        +move()
        +grow(body_part: Tuple[int, int])
        +check_collision() -> bool
    }
    class Food {
        -position: Tuple[int, int]
        +__init__(width: int, height: int)
        +generate_new_position(snake_body: List[Tuple[int, int]])
        +draw()
    }


## Program call flow


sequenceDiagram
    participant M as main.py
    participant G as game.py
    participant S as Snake
    participant F as Food
    M->>G: create_game_instance(width, height)
    G->>S: initialize_snake(start_pos)
    G->>F: initialize_food()
    loop Game Loop
        G->>S: move_snake()
        G->>S: check_collision()
        G->>F: generate_new_position(snake_body)
        G->>G: update_game_state()
        G->>G: draw_screen()
    end
